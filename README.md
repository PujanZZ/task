# Angular-Task

First clone. <br />
```javascript I'm A tab
git clone https://github.com/PujanZZ/task.git
```

Next is to install modules to run the program
```javascript I'm A tab
npm i 
```

After modules installation, ng serve the angular application
```javascript I'm A tab
ng serve
```
